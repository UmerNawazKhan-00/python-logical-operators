# Python Logical Operators

This repository contains simple Python examples explaining the three logical operators:

* **AND**
* **OR**
* **NOT**

These operators are fundamental in programming and are commonly used in conditions and decision-making.

## Code Example

```python
# Logical Operators (AND, OR, NOT)

# NOT
a = 10
b = 5
print(not False)
print(not (a > b))  # a > b is True, but NOT makes it False

# AND
val1 = True
val2 = True
print("AND operator:", val1 and val2)

# OR
val1 = True
val2 = False
print("OR operator:", val1 or val2)
```

## Expected Output

```
True
False
AND operator: True
OR operator: True
```
